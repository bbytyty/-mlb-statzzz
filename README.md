# ⚾ MLB Statzzz

A daily MLB matchup intelligence board inspired by Kasper's model.

## Setup

1. Install requirements:
```bash
pip install -r requirements.txt
```

2. Run locally:
```bash
streamlit run app.py
```

## Deploy to Streamlit Cloud (Free)

1. Push this folder to a GitHub repo
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Set main file path to `app.py`
5. Deploy!

## Features
- 📊 Daily matchup board with color-coded scores
- ⚾ HR Likelihood, Hit Likelihood, Play/Fade ratings
- 🔥 Master composite score
- 📈 All Kasper-style metrics: ISO, xwOBA, SwStr%, PulledBrl%, Brl/BIP%, SweetSpot%
- 🎮 Game-by-game filtering
- 📥 CSV import/export
- 🌙 Dark mode
